Beauty Finder is python programs for crawling data in reviews on Sephora and make customized recommendations to customer.

Beauty Finder is code for crawling reviews on Sephora and making apps that consumers can find url of the highest ranked product that has reviews of whom has same categories of skin type and age range.

Installation& prerequisites:
1-3 python files are in python2.7 language, and the fourth file on app is in python 3.3 language.
1-2 files need internet connection to make results.
